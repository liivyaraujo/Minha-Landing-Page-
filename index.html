Sobre mim

Sou estudante da Universidade Federal do Paraná, atualmente cursando Tecnologia em Análise e Desenvolvimento de Sistemas. Minha trajetória acadêmica começou no Colégio Positivo, onde concluí o ensino médio. Durante o meu último ano do ensino médio e alguns meses de cursinho, comecei a dar aulas particulares de inglês online, chegando a ter até 9 alunos simultaneamente. Assim que fui aprovada no vestibular para o curso de TADS, interrompi as aulas particulares e o cursinho para ingressar na faculdade em junho de 2022.

Durante meus anos na universidade, aprendi muitas coisas sobre tecnologia e me descobri como desenvolvedora web junior. As disciplinas da faculdade me proporcionaram uma base sólida em programação, banco de dados, desenvolvimento web e muitas outras áreas essenciais para o desenvolvimento de sistemas. Essas experiências acadêmicas têm sido fundamentais para meu crescimento profissional, permitindo-me desenvolver habilidades práticas e teóricas que aplico diariamente em meus projetos.

Além disso, fiz parte do centro acadêmico da faculdade, atuando na função de mídias sociais. Nessa posição, gerenciei eventos acadêmicos e colaborei na organização de diversas atividades, o que enriqueceu significativamente meu conhecimento em comunicação e gestão de projetos. Essa experiência foi crucial para desenvolver minhas habilidades de liderança e trabalho em equipe, além de me proporcionar uma compreensão mais profunda sobre a importância da interação e do engajamento dentro do ambiente acadêmico.

Página Pessoal

Este projeto web foi desenvolvido para a disciplina de Desenvolvimento Web 1. Trata-se de uma página pessoal onde elaborei uma breve descrição sobre mim e inclui outras informações relevantes. O objetivo do projeto era aplicar os conhecimentos adquiridos durante o curso e criar uma presença online que refletisse minha trajetória e interesses

Formulário de Cadastro

Este projeto web, desenvolvido para a disciplina de Desenvolvimento Web 1 é um formulário. O objetivo desse projeto era praticar os conhecimentos aprendidos sobre formulários em HTML e em conjunto praticar um pouco de CSS. Essa interface consiste de um exemplo simples de formulário na forma de um cadastro.

Site de receitas

Este projeto foi desenvolvido para a disciplina de Interação Humano-Computador. O objetivo era criar protótipos funcionais de um sistema, e o sistema escolhido foi um site de receitas. O intuito deste projeto foi aprimorar a usabilidade e a experiência do usuário, garantindo que a interface fosse intuitiva e acessível. Trabalhar neste projeto me permitiu aplicar princípios de design centrado no usuário.

Controle de Estoque

Este projeto foi desenvolvido para a disciplina de Análise e Projeto de Sistemas. Ele consiste em um protótipo de um sistema de controle de estoque de um almoxarifado. Além dos protótipos, o trabalho final incluiu toda a documentação necessária do sistema, como diagramas UML, entre outros. O desenvolvimento deste projeto foi fundamental para a disciplina, pois proporcionou uma compreensão prática dos processos de análise e design de sistemas. Trabalhar neste projeto me permitiu aplicar técnicas de modelagem e documentação, essenciais para a criação de sistemas eficazes e bem estruturados. Este exercício foi crucial para desenvolver habilidades em identificar requisitos, projetar soluções e assegurar que todos os aspectos do sistema fossem bem documentados e compreendidos.

Gerenciador de Tarefas

Este projeto de gerenciador de tarefas foi desenvolvido em Java e apresenta uma abordagem moderna para a criação de uma aplicação com interface gráfica de usuário (GUI) utilizando a biblioteca Swing. Diversos conceitos e práticas contemporâneas foram incorporados, resultando em um código modular, eficiente e de fácil compreensão. Desenvolver este gerenciador de tarefas me permitiu aprofundar meus conhecimentos em Java e Swing, além de aplicar boas práticas de programação como a modularização do código.

Data Warehouse

Desenvolvi um projeto de Data Warehouse customizado para uma loja de varejo, com o objetivo de aprimorar a gestão e análise de dados. O projeto envolveu a criação de um modelo Relacional e Lógico de Banco de Dados, correção de inconsistências entre o script SQL e o Diagrama Relacional, e a definição de processos de ETL (Extração, Transformação e Carga). Além disso, foram feitas recomendações de hardware e SGBD (Sistema de Gerenciamento de Banco de Dados), e uma análise gerencial dos benefícios esperados.
